## 🎁 React Shop
- 기존에 있는 React Shop 어플리케이션을 똑같이 구현하는 것을 목표로 한 클론코딩 프로젝트입니다.

<br/>

## 📌구현 기능

- ### dark mode
  테마를 다크모드로 변경할 수 있습니다.
- ### cart
  - 제품의 상세페이지에서 장바구니에 제품을 담을 수 있습니다.
  - 장바구니 페이지에서 수량을 수정할 수 있습니다.
- ### search
  - 제품명으로 검색이 가능하며, 리스트를 클릭해 제품의 상세페이지로 이동이 가능합니다.
  - 검색 시 리스트를 키보드로 접근 가능합니다.
